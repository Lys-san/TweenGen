Inbetween frames generator for animation.

-- ABOUT --

Tween\+Gen is a personnal inbetween frame generator to be used for animation projects. The program uses interpolation methods to calculate the coordinates of a point between two frames, and constructs an inbetween according to the armature of the object to be animated.
